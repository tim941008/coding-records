# 解題紀錄

## 題目：a024. 最大公因數(GCD)

## 📙 題目描述

給定兩個整數，請求出它們的最大公因數


輸入說明
輸入包含兩個整數，以空白鍵隔開，兩個整數均 大於 0, 小於 $2^{31}$

輸出說明
輸出兩個整數的最大公因數

```txt
範例輸入 #1   
12 15

範例輸出 #1 

3
```

```txt
範例輸入 #2
   
1 100

範例輸出 #2

1
```

---

## ✒️ 解題思路

[輾轉相除法](https://zh.wikipedia.org/zh-tw/%E8%BC%BE%E8%BD%89%E7%9B%B8%E9%99%A4%E6%B3%95)

---

## 💻 C++解法

```cpp

#include <bits/stdc++.h>
using namespace std;
class Solution {
   private:
    int a,b;
   public:
    Solution() { cin >> a >> b; }
    void ans(){
        while((a %= b) && (b %= a));
        cout << a + b << endl;
    }
};
int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    Solution s;
    s.ans();
    return 0;
}
```
