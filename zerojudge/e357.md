# 解題紀錄

## 題目：e357. 遞迴函數練習

## 📙 題目描述

定義一個函數 $F(x)$ ，

若 $x = 1$, 則 $F(x) = 1$  
若 $x$ 為偶數，則 $F(x) = F(x/2)$  

其餘狀況， $F(x) = F(x - 1) + F(x + 1)$  

```txt
輸入說明
輸入只有一行，其中包含一個正整數x (1 <= x <= 2000000)。
```

```txt
輸出說明
輸出只有一行，其中包含一個正整數F(x)。
```

```txt
範例輸入 #1    

6

範例輸出 #1 

2
```

---

## ✒️ 解題思路

這題遞迴終止條件為``x`` 等於 ``1``
可以使用自訂函式來根據條件遞迴

---

## 💻 Python解法

```py
def f(n):
   if n == 1:
      return 1
   elif n % 2 == 0:
      return f(n//2)
   else:
      return f(n - 1) + f (n + 1)

if __name__ == '__main__':
   n = int(input())
   print(f(n))
```
